A mono topic pill. Use for post topics, tech tags, and small metadata. Set `href` to link to an archive.

```jsx
<Tag>AI agents</Tag>
<Tag variant="accent" href="/tag/future">the future</Tag>
<Tag variant="bare">2026 · 07</Tag>
```

Variants: `default` (well-tinted chip) · `accent` (rides the live hue) · `bare` (no chip, inline mono label). Keep labels short — one or two words.
