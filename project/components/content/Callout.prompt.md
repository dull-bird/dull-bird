A tinted aside inside an essay — a bird-marked panel for a note, a translation caveat, or a key idea. `accent` tone rides the live hue; `quiet` is neutral gray.

```jsx
<Callout title="呆鸟按">
  笨鸟先飞 —— 与其等技术普及，不如现在就把想法分享出来。
</Callout>

<Callout tone="quiet" mark={false}>
  本文由 AI 翻译，人工校对。English version is machine-translated.
</Callout>
```

Use sparingly — one or two per essay. Drop `mark` for a plain panel.
